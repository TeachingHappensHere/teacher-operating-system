# Teacher Operating System 19.1

This delta connects the Version 19.0 lesson-planning framework to the Version 18 master schedule.

## Features

- Weekly Curriculum Map route and navigation item
- Schedule-aligned Monday–Thursday pacing
- Friday assessment-day pacing
- Cross-curricular weekly alignment summary
- Subject-by-subject lesson readiness
- One-click framework builder that preserves existing content
- Starter objectives and I Do / We Do / You Do lesson structures
- Copy and materials queue with persistent completion tracking
- Responsive phone, iPad, desktop, and print layouts

The framework builder only fills blank lesson fields. It does not overwrite work already entered in Planning Center.
