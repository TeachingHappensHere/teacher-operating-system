### After deployment

1. Hard-refresh the site.
2. Open **Writing, Science & SS**.
3. Build the first Writing lesson.
4. Build the first Science lesson.
5. Build the first Social Studies lesson.
6. Send each lesson to the correct day in Weekly Planning.
7. Add required printable pages to Lesson Attachments.
8. Confirm all three lessons appear in the Daily Lesson Packet.
