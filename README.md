## After deployment

1. Wait for GitHub Pages to finish deploying.
2. Open the site in an incognito/private window.
3. Confirm `First-Week Builder` appears directly below `Classroom Launch`.
4. Open it and select `Generate First Week`.
5. In the normal browser, press Ctrl+Shift+R.
6. If the old page remains, unregister the old service worker or clear cached site files.
