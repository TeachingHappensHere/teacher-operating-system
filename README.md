```html
<script src="launch-candidate.js"></script>
```

### Commit message

`Version 5.0 - Launch Candidate and Installation Center`

### Features

- Guided first-time setup
- Install prompt for supported desktop and Android browsers
- iPad and iPhone Add to Home Screen directions
- Device and browser detection
- Version 5.0 feature inventory
- Launch-readiness checks
- Links to App Health and Backup & Transfer
- First-run onboarding that can be marked complete
- Open My Classroom launch button
- Shortcut: `Ctrl + Shift + L` or `Command + Shift + L`

### After deployment

1. Refresh the live GitHub Pages site.
2. Open Launch Center.
3. Complete the setup checklist.
4. Run App Health.
5. Create a backup.
6. Install the app.
7. Open Teach My Day from the installed application.
