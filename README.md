# Version 19.0 — Connected Lesson Planning Foundation

This release integrates the lesson-planning framework into the working Teacher Operating System.

## Included
- Planning Center navigation
- Connected Weekly Hub centered on the Open Court anchor text
- Shared lesson model for Reading, Phonics, Heggerty, Tier II, Writing, Math, Science, and Social Studies
- Monday–Friday Lesson Studio
- Arizona standards field
- Essential question and success criteria
- I Do / We Do / You Do framework
- Checks for understanding, assessment, differentiation, accommodations, notes, and reflection
- Lesson Ready status and weekly readiness calculations
- Duplicate Week workflow
- Print view
- Planbook-compatible text export
- Local autosave on the current device
- Responsive phone and iPad layout

## Seeded instructional week
The first core week begins August 3, 2026 and is seeded with Open Court Unit 1, Lesson 1, “The Mice Who Lived in a Shoe.”
