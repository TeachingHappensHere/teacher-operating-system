WHAT IS NEW
- New categorized Resource Hub for daily teaching platforms.

- Add or edit website links directly in the app.
- Resource links save on the current device.
- Filter resources by category.
- Responsive desktop, iPad, and phone layout.

GITHUB INSTALLATION
Upload all five files to the repository root. Choose "Commit changes." GitHub will replace index.html and service-worker.js and add the new Resource Hub files.

After deployment, refresh the site twice so the updated service worker activates.
